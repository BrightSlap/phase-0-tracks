What is the purpose of HTML5 semantic tags? Which ones do you see as most useful?

The purpose of the semantic tags is to display the contents of the body of the page in the format that is desired by the developer and allow for fluidity. I see nav, aside, article, and sections as most useful.

What is the order the browser uses to display elements on a page?

The order the elements are displayed through the browser is top to bottom.
